Data Structures

A data structure is simply a storage format for data. A data structure stores data within it along with possible relations between the data as well as functions and operations that can be applied to the data. Different types of data structures are better for different kinds of applications.


Arrays

Arrays are data structures that store a collection of elements, typically of the same data type, in contiguous memory locations. Each element is accessed using an index, starting from zero. 


            // declare an array of 4 integers
            int myNumbers[4] = {25, 50, 75, 100};
            // print the value at the 0th index of the array
            printf("%d", myNumbers[0]);
            // outputs 25

Array Algorithms

You will use arrays a lot in your programming adventures. And since they are so ubiquitous, they also tend to make use of many different algorithms. Below is a short list of some of the most frequently used algorithms when working with arrays.

  • Linear Search
  • Binary Search
  • Bubble Sort
  • Selection Sort
  • Merge Sort
  • Quick Sort
  • Sliding Window
  • Kadane's Algorithm

This list is, of course, not exhaustive. And I will update this page in the future with more algorithms that I write posts regarding.
